Set a new remaining time.

If timeFlowScreen is shown, it will be updated on the lcd _lcd else it will be memorized for when it will be shown. If waitRefresh is true (default is true) it will wait LCDREFRESHTIME mSec before updating _lcd.

Parameters
timeSec: unsigned int. current CV between 0 to ssrWindow
waitRefresh: boolean. ssrWindow size in mSec.
