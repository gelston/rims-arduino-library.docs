Detailed Description

Library to menage LCD and keypad for Rims library.

Made for : DFRobot LCD Keypad Shield For Arduino http://www.dfrobot.com/index.php?route=product/product&product_id=51#.Ua1dHhXEnXQ

If anything else is used, lcd must be 16 cols 2 rows and keypad must be associated to a analog ADC pin with a volage divider for each keys. Needed keys : UP, DOWN, LEFT, RIGHT, SELECT

Author
Francis Gagnon
